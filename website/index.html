SPACEMFERS: mfers in space

spacemfers is a work-in-progress that aims to be a browser-based, open-universe, space sandbox mmo where mfers (like you) can explore the mysteries of space, together! currently in development by zhoug (me).

it's being built on top of the farcaster id system so you can login w/ warpcast. that way, the game can also read your profile details & any connected crypto addresses too.

starting small & making things up as i go, check twitter for updates!

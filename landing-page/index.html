SPACEMFERS: mfers in space

spacemfers is a work-in-progress that aims to be a browser-based, open-universe, idle-clicker, lofi space sandbox mmo where mfers can explore the mysteries of space together, currently in active development by zhoug (me).

(i'm thinking a neopets meets habbo hotel meets eve online kinda vibe)

it's being built on top of the farcaster id system, so you'll be able to login w/ warpcast. the game can then read your profile details & any connected crypto addresses wooo.

starting small & making things up as i go, check twitter for updates!
